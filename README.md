# search-algolia

> An **Algolia** driver for [togo](https://to-go.dev) full-text search.

Registers the `algolia` driver on the [`search`](https://github.com/togo-framework/search) plugin.

## Install

```sh
togo install togo-framework/search-algolia
```

Then in `.env`:

```sh
SEARCH_DRIVER=algolia
ALGOLIA_APP_ID=your-app-id
ALGOLIA_API_KEY=your-admin-api-key
```
